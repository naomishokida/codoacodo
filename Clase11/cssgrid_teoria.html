CSS GRID: Teoría

CSS GRID:

  • Es un modelo bidimensional de layout basado en cuadrículas, se usa para dividir una página en pequeños y espaciados segmentos.
  • Nos permite manejar las filas y columnas a nuestro gusto.
  • La diferencia con Flexbox es que es unidimensional, cuando íbamos agregando flex-items se iban posicionando uno al lado del otro en el eje principal. En cambio Grid los va ubicando tanto en filas como en columnas.
  • Para empezar, necesitamos definir un container, el cual actuará como elemento padre y en el cual tendremos los grid-items. A este elemento le asignaremos en CSS: .container { display:grid; }

Elementos:

  • Grid-lines: son las líneas de nuestra cuadrícula.
  • Grid-items.
  • Grid-cells.
  • Grid-track: representa una fila con una columna.
  • Grid-area: Representa un espacio rectangular en nuestra grilla.

Grid Templates

Grid Templates: Nos permite crear filas o columnas a nuestro gusto o necesidad.

  • grid-template-columns: columna1 columna2 columna3;
  • grid-template-rows: fila1 fila2 fila3;
    • Podemos abreviar esto con:
  • grid-template: < rows > / < columns >

Repeat

Si las columnas o filas son iguales, podemos usar esta función para no tener que repetir manualmente. Nos permite repetir patrones para definir filas o columnas.
repeat(< veces a repetir >, < patron >)
grid-template-rows: repeat(6,150px);

Unidad fr

Unidad de medida únicamente para los Grid, indica el espacio disponible que tenemos en la grilla. Es más responsive. Representa una fracción. o tendrías que usar ningún media-query.

Grid y flex trabajan juntos, son complementarios. Lo ideal para empezar a desarrollar es una base de layouts o división de columnas que le podamos aplicar a los divs.

Asignar espacios

Son propiedades de los grid-items:

  • grid-column-start: desde qué linea en columna va a ocupar el item.
  • grid-column-end: Hasta qué linea en columna va a ocupar el item.
  • grid-column: < grid-column-star / < grid-column-end >
  • grid-row-start: Desde qué linea en fila va a ocupar el item.
  • grid-row-end: Hasta qué linea en fila va a ocupar el item.
  • grid-row: < grid-row-star / < grid-row-end >
