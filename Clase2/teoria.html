Estructura básica

  • !DOCTYPE HTML: Para saber con qué versión de HTML estamos trabajando.
  • html lang="es": Etiqueta que va a englobar a todo el resto. Siempre acompañada del atributo lang.
  • head: Acá van los meta, links, título de la pág.
  • meta charset="UTF-8": Caracteres especiales
  • meta name="viewport" content="width=device-width, initial-scale=1.0": Relacionada al responsive de la página
  • title: Nombre de la página
  • link rel: Para llamar a los archivos externos que vamos a utilizar
  • body: Acá va a estar todo el contenido audiovisual de la página

Etiquetas semánticas

  • header: Encabezado de un documento o sección. Su fin es que lo utilicemos como un contenedor introductorio. Título principal, nombre de la pág, logo.
  • nav: En algunos casos se puede incluir en el Header, define un conjunto de enlaces de navegación.
  • main: Representa al contenido principal dentro del body. No va a tener información.
  • section: Define secciones de un documento. Capítulos, encabezados. No debería utilizarse como un contenedor genérico, para esto se utiliza div. Tiene que tener secciones que sean relevantes.
  • article: Especifica contenido autónomo. Mensajes de un foro, comentarios.
  • aside: No es muy utilizada, aunque se usa para colocar información adicional. Publicidad.
  • footer: Pie de pagina, está relacionado con el copyright, mapa de un sitio, información de contacto.

Estas etiquetas son las que le dan un significado a nuestro documento.

No todas las etiquetas tienen atributos. Algunas etiquetas sí tienen etiquetas y valores.

Enlaces o hipervínculos

  • a: Enlaces o hipervínculos. Siempre va acompañado del atributo href, que indica el enlace al que uno quiere ir. Atributo target="_blank": permite que lo abra en otra nueva pestaña. Hay tres tipos de enlace:
    • Absolutos: Incluye todas las partes de una URL.
    • Relativos: Lo vamos a llamar desde carpetas locales.
    • Ancla: Es para ir a un lugar dentro de la misma página.

Imágenes

No tiene etiqueta de cierre. Va con el atributo src="". Tiene de atributo obligatorio alt="". Se puede agregar width y height. Mediante css se puede agregar imágenes también.
