About Me

Hello, I'm a frontend developer from Buenos Aires, Argentina. I have a bachelor's degree in architecture from the 'University of Buenos Aires'. But about two years ago I've begun to learn web development and programming. The first course I took about web design was the 'Professional Frontend Developer' course at the 'National Technological University UTN FRBA'. That's when I realized that I really like this because I can combine my knowledge in design with code! So nowadays I still keep learning to improve my skills and be a better frontend developer. I’m currently interested in learning more about JavaScript and Java to become a full stack web developer.
